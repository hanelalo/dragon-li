# 01. 产品目标与范围

## 项目命名

- 中文名：狸花猫
- 英文名：Dragon-Li

## 核心目标

- 做一个通用 LLM 对话应用，支持对接多家 API。
- 提供会话管理能力（创建、切换、归档、检索）。
- 支持短期记忆与长期记忆。
- 记忆以 Markdown 文件承载，便于可读、可编辑、可迁移。

## 产品形态

- 以可视化界面为主。
- 最终形态为桌面应用。

## MVP 边界

- Provider 仅支持 OpenAI 与 Anthropic。
- 跑通对话链路、记忆审核写入、基础检索。
- 暂不做旧目录迁移策略。
- 首版禁用外部工具调用。
- 首版仅允许 Agent 访问 `~/.dragon-li/` 数据域，不读取工程目录。
